About

ZeroBooks is a software project for the buying and selling of books.

What you can do on ZeroBooks:
  • View the books
  • Upload new books
  • Search for books, authors, e.t.c.
ZeroBooks is powered by these tools:
  • Python
  • Django
  • Bootstrap
  • PostgreSQL
  • DjangoRestFramework
  • Django-Rest-Auth
  • StartBootstrap
